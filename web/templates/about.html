About Geotiny Seismic Monitor

Educational platform for learning seismology and geophysics

Project Overview
Geotiny Seismic Monitoring Network

This project is part of the Interdisciplinary Transformational University initiative in Austria, providing students with hands-on experience in seismology, data acquisition, and signal processing.

The Geotiny network consists of 3 professional MEMS-based seismic sensors deployed across campus, providing real-time monitoring of local seismic activity and integration with global earthquake data.

Getting Started
Dashboard

The dashboard shows real-time waveforms from all 3 devices with current amplitudes and buffer statistics.

Spectrum Analysis

Select a device and time window (1, 10, or 30 minutes) to analyze the frequency content:

  • FFT Spectrum: Shows dominant frequencies in the signal
  • Spectrogram: Visualizes how frequency content changes over time
  • Time Domain: The raw waveform data
Global Seismic

Monitor worldwide earthquake activity with filters by magnitude and time period. Click on events to see details and locations on an interactive map.

Key Concepts
  • FFT: Fast Fourier Transform converts time to frequency domain
  • Nyquist Frequency: Maximum resolvable freq = fs/2 = 50 Hz
  • Spectrogram: 2D plot of frequency vs time
  • Sampling Rate: 100 Hz = 100 samples per second
