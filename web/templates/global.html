About Global Earthquake Data

This data comes from the USGS Earthquake Hazards Program, which monitors earthquake activity worldwide.

  • Data is updated in real-time
  • Includes all recorded earthquakes above the selected magnitude threshold
  • Location accuracy depends on seismic network density in each region
  • For research-grade data, visit earthquake.usgs.gov
